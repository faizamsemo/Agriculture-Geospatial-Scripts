# Agriculture Geospatial Scripts

A comprehensive collection of Python and Google Earth Engine (JavaScript) scripts for agricultural monitoring using GIS, Remote Sensing, Machine Learning, and Earth Observation data.

## Features

- Crop monitoring
- Vegetation indices
- Drought assessment
- Soil moisture analysis
- Irrigation planning
- Crop classification
- Yield prediction
- Weather analysis
- Time-series analysis
- Machine learning models

## Programming Languages

- Python
- Google Earth Engine JavaScript

## Data Sources

- Sentinel-2
- Landsat
- MODIS
- CHIRPS
- ERA5
- SMAP
- SoilGrids
- Dynamic World
